Python Snake Game

Python Snake Game was Nahyun and Ashlyn’s first ever hackathon project. It was an educational snake game coded with Python. By incorporating maths questions into the game, and only allowing the user to progress once they get the question correct. Nahyun and Ashlyn aimed to create a fun learning tool for primary school students, improving education technology.

VR Circus Simulator

Ashlyn knew from the moment she saw a clown for the first time (at the local Macca’s) that the circus was where she belonged. Pressured into entering a commerce career, she chose to be an actuary by day, circus master by night - by creating a virtual reality open sandbox game set in a circus. In the game, you are an uncoordinated, unemployed orphaned child adopted by the head clown of a travelling circus. You make the decisions in the game - whether it is training yourself to become a world-famous juggling trapeze artist, rising up in the circus ranks, befriending the lions, or performing a daring escape from the circus at night in a quest to find your family…

Co-developer of Pingu Scholar

The next initiative for Ashlyn while Nahyun creates PenGoogle is to create a reputable database for any information about Penguins. An extension of Pengoogle, Pingu Scholar proves to be an invaluable resource for any penguin lover.

Scribble It

In a world where people come from all ages and all backgrounds, handwriting can be a mess. Scribble It is a third-party software designed to read the handwriting of all kinds, doctors to teachers, children to the elderly. With Scribble It, Ashlyn aims to bridge the boundaries between handwriting, allowing everyone to be comfortable with their own unique handwriting.
